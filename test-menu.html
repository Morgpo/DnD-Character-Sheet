If it works here: The issue is with the main HTML file.

If it doesn't work: Your browser might be blocking JavaScript.

Troubleshooting Steps:

  1. Check browser console: Press F12, go to Console tab, look for errors
  2. Try a different browser: Test in Chrome, Firefox, or Edge
  3. Enable JavaScript: Make sure JavaScript is enabled in your browser settings
  4. Run a local server: Some browsers block local file JavaScript. See below.

To Run a Local Server (Recommended):

If you have Python installed:

# Open PowerShell in the project folder, then run:
python -m http.server 8000

# Then open browser to: http://localhost:8000

If you have Node.js installed:

# Install http-server globally:
npm install -g http-server

# Then in the project folder:
http-server

# Open browser to: http://localhost:8080

Or use VS Code:

# Install "Live Server" extension in VS Code
# Right-click index.html and select "Open with Live Server"
